Q1: ဇာတ်ကားတွေက save လို့မရဘူးလား။ +

VIP Channel ကဇာတ်ကားတွေက save လို့မရပါဘူး။ ဖုန်းထဲသိမ်းမရပါ။ Save ဖွင့်လိုက်ရင် ဇာတ်ကားတွေပျက်တာမလို့ ဘယ်သူ့ကိုမှ ပေးဖွင့်မရပါဘူးခင်ဗျာ။

Q2: TV နဲ့ချိတ်ကြည့်လို့မရဘူးလား။ +

TV အမျိုးအစားပေါ်မူတည်ပြီး ကြည့်ရတဲ့လူတွေရပါတယ်။ တချို့လည်းကြည့်မရပါ။ Admin တို့လည်းဘာမှလုပ်ပေးမရပါဘူးဗျ။

Q3: ဇာတ်ကားအသစ်တွေမတင်ဘူးလား။ +

ရုံတင်ကားတွေက တစ်လကို (၁) ကား၊ (၂) ကား၊ အလွန်ဆုံးမှ (၃) ကားလောက်ပဲထွက်ပါတယ်။

မြန်မာကားတွေက တခြားကားတွေလို အများကြီးမထွက်ပါဘူး။

Q4: မထွက်သေးတဲ့ ရုံကြီးကားတွေ ဘယ်တော့လောက်ကြည့်လို့ရမလဲ။ +

လောလောလတ်လတ်ရုံတင်နေတဲ့ကားတွေ ရုံတင်ပြီးသားဇာတ်ကားကြီးတွေ ဘယ်တော့ထွက်မလဲဆိုတာ Admin တို့မသိနိုင်ပါဘူး။ ချန်နယ်လိုင်းတွေကနေပြမှပဲ ကြည့်လို့ရပါတယ်။ ရုံတင်ပြီး အနည်းဆုံး ၂ နှစ်ကနေ ၅ နှစ်အတွင်း ထွက်နိုင်ပါတယ်။ တချို့ကားတွေလည်း လုံးဝမထွက်ပါဘူး။

Q5: ဇာတ်ကားတွေက ကြည့်မရဘူး။ ရပ်သွားတယ်။ ပွင့်မလာဘူး။ +

အဲ့ဒါကတော့ ကိုယ်အသုံးပြုနေတဲ့ ဖုန်းနဲ့ဆိုင်တာမလို့ ကျွန်တော်တို့ပြင်လို့မရပါဘူးဗျ။

ဇာတ်ကားတွေက ကျွန်တော်တို့ဘက်ကနေ ဖုန်းနဲ့ကြည့်ပြီး ကြည့်ရတယ်ဆိုမှ တင်တာပါ။

မန်ဘာတော်တော်များများလည်း ကြည့်လို့ရပါတယ်။ ဗီဒီယိုဖိုင်အမျိုးအစားနဲ့ ဖုန်းနဲ့မကိုက်တာမလို့ ဖြစ်နိုင်ပါတယ်။

ကျွန်တော်တို့ပြင်ပေးလို့ မရနိုင်တာကိုတော့ နားလည်ပေးပါဗျ။

Q6: Telegram အကောင့်ပြောင်းဝင်လို့ရလား။ +

အခု မိမိဝယ်ထားတဲ့အကောင့်ကလွဲရင် တခြားအကောင့်ကို ပြောင်းသုံးလို့မရပါဘူး။ လက်ရှိ VIP ပေါက်ဈေးနဲ့ မန်ဘာအသစ်ထပ်ဝင်မှ ရပါမယ်ခင်ဗျ။
